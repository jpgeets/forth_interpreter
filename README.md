# Forth Interpreter

Small experimental Forth-like interpreter written in Python.

This repository is intended as a compact codebase for AI training experiments (agentic coding workflows).

The project is intentionally kept lightweight so that it can easily be extended, tested, refactored, and adapted to different experimentation scenarios involving AI-assisted code generation and reasoning.

## Current Features

- stack-based execution model
- token parsing
- basic arithmetic operators
- pytest-based test suite
